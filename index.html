Welcome to Creative Coding

Learn to create art through code using JavaScript and p5.js.

Instructor and Curricular Design : Matthew Nolan, University of Maryland, Department of Computer Science.

Course Overview

This 15-week course introduces you to creative coding through hands-on projects. You'll learn programming fundamentals while creating visual art, interactive installations, data visualizations, and more.

No prior coding experience required! This course is designed for artists and designers with mixed technical backgrounds.

Meet Your Lecturer Matt Nolan

Welcome to Creative Coding! I'm excited to guide you through this journey of exploring code as a creative medium. This course is designed to help you discover the expressive possibilities of programming, whether you're completely new to code or looking to expand your creative practice.

Throughout this course, you'll learn to think like a creative coder—using programming concepts to generate visual art, create interactive experiences, and express your artistic vision through code. We'll work with p5.js, a powerful and accessible creative coding library that makes it easy to bring your ideas to life.

My goal is to create a supportive learning environment where experimentation is encouraged, mistakes are part of the process, and your unique creative voice can emerge. Don't hesitate to reach out with questions, share your work, or discuss ideas. I'm here to help you succeed.

Contact: mttnln@umd.edu

Course Structure

Weeks 1-5: Foundations

Building the basics and seeing code as a creative tool

Weeks 6-8: Building Complexity

Working with collections and creating sophisticated systems

Weeks 9-10: Sound and UI

Working with audio, sound synthesis, and creating interactive UI controls

Weeks 11-12: Interactivity and Data

Game design, state machines, scoring systems, and data visualization

Weeks 13-15: Video, Studio Work, and Portfolio

Video input and processing, independent studio work, and final portfolio development
