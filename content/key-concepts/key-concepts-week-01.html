Key Concepts

What is Creative Coding?

Creative coding treats code as instructions for the computer. It differs from traditional programming in its focus on creative expression. p5.js serves as a tool for artists, providing a creative coding workflow that emphasizes experimentation and visual output.

Basic p5.js Structure

Code example 1 written in javascript. The code follows.
function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  // Your drawing code here
}

The setup function runs once when the program starts. The draw function runs continuously, creating animation. The createCanvas function sets the size of your drawing area.

Using console.log() for Debugging

console.log() is one of the most important tools for learning and debugging code. It lets you see values, check if code is running, and understand what your program is doing. The output appears in the browser's Developer Console, not on your canvas.

How to Open the Console:

  • p5.js Web Editor (editor.p5js.org): Click the "Console" button at the bottom of the editor, or press Ctrl+Shift+K (Windows/Linux) or Cmd+Option+K (Mac)
  • OpenProcessing.org: Press F12 to open Developer Tools, then click the "Console" tab
  • Browser (local HTML): Press F12 to open Developer Tools, then click the "Console" tab, or right-click → "Inspect" → "Console" tab

Basic Usage:

Code example 1b written in javascript. The code follows.
// Log a message
console.log("Setup is running!");

// Log a variable value
let x = 200;
console.log("X position:", x);

// Log multiple values
console.log("Canvas size:", width, "x", height);

When to Use console.log():

  • Understanding code flow: See which functions run and when
  • Checking variable values: See what numbers, strings, or other values your variables contain
  • Debugging problems: When something isn't working, log values to see what's wrong
  • Learning new functions: Log the output of functions to understand what they return
  • Tracking changes: See how values change over time

Important Tips:

  • Use descriptive messages: console.log("Circle X position:", circleX); is better than just console.log(circleX);
  • Don't log in draw() without conditions: The draw() function runs 60 times per second. Logging every frame floods the console. Use conditions like if (frameCount % 60 === 0) to log less frequently
  • Log before and after changes: This helps you see what's happening to your variables
Code example 1c written in javascript. The code follows.
function draw() {
  background(220);
  
  // Log only on first frame
  if (frameCount === 1) {
    console.log("Draw started");
  }
  
  // Log every 60 frames (once per second)
  if (frameCount % 60 === 0) {
    console.log("Frame:", frameCount, "Mouse:", mouseX, mouseY);
  }
  
  // Log only when mouse is pressed
  if (mouseIsPressed) {
    console.log("Mouse clicked at:", mouseX, mouseY);
  }
}

Common Patterns:

  • Check if code is running: console.log("This function ran");
  • See variable values: console.log("myVariable is:", myVariable);
  • Check calculations: console.log("Distance:", dist(x1, y1, x2, y2));
  • Track execution: console.log("Step 1:", value1, "Step 2:", value2);

Coordinate System

The origin at coordinates 0, 0 is at the top-left corner of the canvas. The X coordinate increases as you move right. The Y coordinate increases as you move down. For example, to draw at the top-left, use small x and y values such as 10, 10.

Basic Drawing Functions

p5.js provides several functions for drawing shapes. Each shape is drawn in the order it appears in your code, so shapes drawn later appear on top of earlier ones.

Ellipse:
The ellipse() function draws circles and ovals. It takes four parameters: x position, y position, width, and height. When width and height are equal, you get a circle. When they differ, you get an oval.

Code example 2 written in javascript. The code follows.
ellipse(200, 200, 100, 100);  // Circle at center
ellipse(100, 100, 80, 120);   // Oval (taller than wide)

Rectangle:
The rect() function draws rectangles. It uses the same four parameters: x, y, width, and height. By default, rectangles are drawn from the top-left corner.

Code example 3 written in javascript. The code follows.
rect(100, 100, 200, 150);  // Rectangle: x=100, y=100, width=200, height=150

Line:
The line() function draws straight lines. It takes four parameters: x1, y1 (start point) and x2, y2 (end point).

Code example 4 written in javascript. The code follows.
line(0, 0, 400, 400);      // Diagonal line from top-left to bottom-right
line(100, 200, 300, 200); // Horizontal line

Triangle:
The triangle() function draws triangles. It takes six parameters: three pairs of x, y coordinates for the three vertices.

Code example 5 written in javascript. The code follows.
triangle(200, 50, 150, 150, 250, 150);  // Triangle pointing up

Point:
The point() function draws a single pixel. It takes x and y coordinates.

Code example 6 written in javascript. The code follows.
point(200, 200);  // Single pixel at center

Text:
The text() function displays text on the canvas. It takes the text string and x, y coordinates. Use textSize() to set the size and textAlign() to control alignment.

Code example 7 written in javascript. The code follows.
textSize(32);
textAlign(CENTER);
text("Hello World", 200, 200);

Colors

Colors in p5.js control how shapes appear. You set colors before drawing shapes, and the color settings apply to all shapes drawn afterward until you change them.

Background:
The background() function sets the canvas background color. It clears the entire canvas and sets the base color. You typically call it at the start of draw() to clear previous frames.

Code example 8 written in javascript. The code follows.
background(220);           // Light gray background
background(255, 0, 0);    // Red background
background(0);            // Black background

Fill:
The fill() function sets the interior color of shapes. Once set, all subsequent shapes use this fill color until you change it.

Code example 9 written in javascript. The code follows.
fill(255, 0, 0);  // Red fill
ellipse(100, 100, 50, 50);  // Red circle
fill(0, 255, 0);  // Green fill
ellipse(200, 200, 50, 50);  // Green circle

Stroke:
The stroke() function sets the outline color of shapes. Like fill, it applies to all shapes drawn afterward.

Code example 10 written in javascript. The code follows.
stroke(0);  // Black outline
strokeWeight(3);  // 3-pixel thick outline
ellipse(200, 200, 100, 100);

No Fill / No Stroke:
Use noFill() to draw shapes without interior color (transparent fill). Use noStroke() to draw shapes without outlines.

Code example 11 written in javascript. The code follows.
noFill();
stroke(0);
ellipse(200, 200, 100, 100);  // Circle with only outline, no fill

Color Formats:

  1. Grayscale: Single number from 0 (black) to 255 (white)

    Code example 12 written in javascript. The code follows.
    fill(128);  // Medium gray

  2. RGB: Three numbers for red, green, blue (each 0-255)

    Code example 13 written in javascript. The code follows.
    fill(255, 0, 0);    // Red
fill(0, 255, 0);     // Green
fill(0, 0, 255);     // Blue
fill(255, 255, 0);   // Yellow (red + green)

  3. Named Colors: String values (limited set)

    Code example 14 written in javascript. The code follows.
    fill('red');
fill('blue');
fill('yellow');

Color Order Matters:
Shapes are drawn in the order they appear in your code. Later shapes appear on top of earlier ones, creating layers.

Important Functions to Know

Function Purpose Example
createCanvas(w, h) Creates drawing area createCanvas(400, 400)
background(color) Sets background color background(220) or background(255, 0, 0)
fill(color) Sets fill color for shapes fill(255, 0, 0) or fill('red')
stroke(color) Sets outline color stroke(0) for black outline
noFill() Removes fill (transparent) noFill()
noStroke() Removes outline noStroke()
strokeWeight(size) Sets outline thickness strokeWeight(3)
ellipse(x, y, w, h) Draws circle/oval ellipse(200, 200, 100, 100)
rect(x, y, w, h) Draws rectangle rect(100, 100, 200, 150)
line(x1, y1, x2, y2) Draws line line(0, 0, 400, 400)
triangle(x1, y1, x2, y2, x3, y3) Draws triangle triangle(200, 50, 150, 150, 250, 150)
point(x, y) Draws single pixel point(200, 200)
text(str, x, y) Draws text text("Hello", 200, 200)
textSize(size) Sets text size textSize(32)
textAlign(align) Sets text alignment textAlign(CENTER)
console.log(value) Prints values to browser console for debugging console.log("Hello"); or console.log("X:", x);

Common Patterns

Drawing Multiple Shapes

Code example 15 written in javascript. The code follows.
function draw() {
  background(220);
  fill(255, 0, 0);
  ellipse(100, 100, 50, 50);
  fill(0, 255, 0);
  rect(200, 200, 100, 100);
}
