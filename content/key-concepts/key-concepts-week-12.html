Key Concepts

Loading External Data

p5.js can load data from files (JSON, CSV) or APIs. This lets you work with real-world datasets, create data visualizations, and build data-driven art projects.

Loading JSON Files:
JSON (JavaScript Object Notation) stores structured data as objects and arrays. It's perfect for hierarchical data.

Code example 1 written in javascript. The code follows.
let data;

function preload() {
  // Load JSON file (must be in sketch folder or accessible URL)
  data = loadJSON('data.json');
}

function setup() {
  createCanvas(400, 400);
  console.log(data);
  // Access data properties
  // If JSON is: {"name": "Value", "count": 10}
  // Access with: data.name, data.count
}

JSON Structure Examples:

Code example 2 written in javascript. The code follows.
// Simple object
{"temperature": 72, "humidity": 45}

// Array of objects
[
  {"name": "Item 1", "value": 10},
  {"name": "Item 2", "value": 20}
]

// Nested structure
{
  "location": {"city": "New York", "population": 8000000},
  "data": [1, 2, 3, 4, 5]
}

Loading CSV Files:
CSV (Comma-Separated Values) stores tabular data. Perfect for spreadsheets, sensor data, or any row-column data.

Code example 3 written in javascript. The code follows.
let table;

function preload() {
  // Load CSV with header row
  // Parameters: filename, format, header option
  table = loadTable('data.csv', 'csv', 'header');
}

function setup() {
  createCanvas(400, 400);
  
  // Access table data
  for (let row of table.rows) {
    // Get value by column name (if header row exists)
    let name = row.get('name');
    let cellValue = row.get('value');

    // Or by column index
    let firstCol = row.get(0);
    let secondCol = row.get(1);
  }
}

Data Visualization

Data visualization transforms numerical data into visual representations, making patterns and relationships easier to understand. Effective visualizations help tell stories with data.

Visual Encoding:
Map data values to visual properties:

  • Position: X, Y coordinates
  • Size: Width, height, radius
  • Color: Brightness, hue, saturation
  • Shape: Different shapes for different categories

Basic Bar Chart:

Code example 6 written in javascript. The code follows.
let data = [10, 20, 30, 40, 50];

function setup() {
  createCanvas(400, 400);
}

function draw() {
  background(220);
  
  let barWidth = width / data.length;
  
  for (let i = 0; i < data.length; i++) {
    // Map data value to bar height
    let barHeight = map(data[i], 0, 50, 0, height);
    
    // Draw bar
    rect(i * barWidth, height - barHeight, barWidth - 2, barHeight);
  }
}

Scatter Plot:

Code example 7 written in javascript. The code follows.
let data = [
  {x: 10, y: 20},
  {x: 30, y: 40},
  {x: 50, y: 30}
];

function draw() {

Line Chart:

Code example 8 written in javascript. The code follows.
let data = [10, 25, 30, 45, 50, 35, 40];

function draw() {
  background(220);
  stroke(0);
  noFill();
  beginShape();
  for (let i = 0; i < data.length; i++) {
    let x = map(i, 0, data.length - 1, 0, width);
    let y = map(data[i], 0, 50, height, 0);
    vertex(x, y);
  }
  endShape();
}

Generative Systems

Generative systems create art through rules, algorithms, and processes. The artist defines the system, and the system generates the artwork. This creates a collaboration between human intention and algorithmic process.

Key Principles:

  1. Rules-Based Generation:

    • Define clear rules for how elements are created
    • Rules determine placement, size, color, relationships
    • Same rules can produce infinite variations

  2. Emergent Behavior:

    • Complex patterns emerge from simple rules
    • Interactions between elements create unexpected results
    • System behavior isn't always predictable

  3. Iteration and Variation:

    • Repeat processes with slight variations
    • Use loops, randomness, and noise for variation
    • Each run produces different but related results

  4. Parameter Control:

    • Adjustable parameters change system behavior
    • Allows exploration of parameter space
    • Find interesting combinations through experimentation

Example Generative System:

Code example 10 written in javascript. The code follows.
function setup() {
  createCanvas(400, 400);
  noLoop();  // Static generative art
}

function draw() {
  background(220);
  
  // Rule: Draw circles in grid
  // Variation: Size based on noise
  for (let x = 0; x < width; x += 20) {
    for (let y = 0; y < height; y += 20) {
      // Parameter: noise scale
      let circleSize = noise(x * 0.01, y * 0.01) * 30;
      ellipse(x, y, circleSize, circleSize);
    }
  }
}

Interactive Generative System:

Code example 11 written in javascript. The code follows.
function draw() {
  background(220);
  
  // Rule: Draw lines from center
  // Variation: Angle and length change over time
  for (let i = 0; i < 20; i++) {
    let angle = (TWO_PI / 20) * i + frameCount * 0.01;
    let length = 50 + sin(frameCount * 0.05 + i) * 30;
    let x2 = width/2 + cos(angle) * length;
    let y2 = height/2 + sin(angle) * length;
    line(width/2, height/2, x2, y2);
  }
}

Common Patterns

Visualizing JSON Data

Code example 12 written in javascript. The code follows.
let data;

function preload() {
  data = loadJSON('data.json');
}

function draw() {
  background(220);
  for (let i = 0; i < data.values.length; i++) {
    let dataValue = data.values[i];
    let x = map(i, 0, data.values.length, 0, width);
    let h = map(dataValue, 0, 100, 0, height);
    rect(x, height - h, width/data.values.length, h);
  }
}

Generative Pattern System

Code example 13 written in javascript. The code follows.
function draw() {
  background(220);
  for (let x = 0; x < width; x += 20) {
    for (let y = 0; y < height; y += 20) {
      let noiseValue = noise(x * 0.01, y * 0.01);
      let circleSize = map(noiseValue, 0, 1, 5, 20);
      ellipse(x, y, circleSize, circleSize);
    }
  }
}

Data-Driven Color

Code example 14 written in javascript. The code follows.
let data = [10, 30, 50, 70, 90];

function draw() {
  background(220);
  for (let i = 0; i < data.length; i++) {
    let brightness = map(data[i], 0, 100, 0, 255);
    fill(brightness);
    rect(i * 100, 100, 80, 80);
  }
}
