Key Concepts

Random Function

The random() function generates random numbers. It's essential for creating variation, unpredictability, and organic-looking results in your sketches.

Basic Usage:

Code example 1 written in javascript. The code follows.
random(10);        // Random number from 0 to 10 (not including 10)
random(5, 15);     // Random number from 5 to 15 (not including 15)
random(0, 255);    // Random number from 0 to 255 (for colors)

Important: random() returns a floating-point number (decimal). The upper bound is not included:

  • random(10) returns 0.0 to 9.999... (never exactly 10)
  • random(5, 15) returns 5.0 to 14.999... (never exactly 15)

Common Uses:

  1. Random Positions:
Code example 2 written in javascript. The code follows.
ellipse(random(width), random(height), 50, 50);
  1. Random Colors:
Code example 3 written in javascript. The code follows.
fill(random(255), random(255), random(255));  // Random RGB
  1. Random Sizes:
Code example 4 written in javascript. The code follows.
let circleSize = random(10, 100);
ellipse(200, 200, circleSize, circleSize);
  1. Random Speeds:
Code example 5 written in javascript. The code follows.
let moveSpeed = random(-3, 3);  // Random speed in any direction
x += moveSpeed;
  1. Random Choices:
Code example 6 written in javascript. The code follows.
let colors = ['red', 'blue', 'green'];
let randomColor = colors[floor(random(colors.length))];

Random Seed:
By default, random() produces different results each time. Use randomSeed() to get the same sequence:

Code example 7 written in javascript. The code follows.
randomSeed(42);  // Same seed = same random sequence

Random Walk

Objects that move randomly create organic patterns.

Code example 8 written in javascript. The code follows.
let x = width/2;
let y = height/2;

function draw() {
  stroke(0);
  point(x, y);
  x += random(-2, 2);
  y += random(-2, 2);
}

Noise Function

The noise() function generates smooth, continuous random values using Perlin noise. Unlike random(), which produces completely independent values, noise() creates values that smoothly transition from one to the next, resulting in natural, organic-looking patterns.

Why Use Noise Instead of Random:

  • Smooth transitions: Values change gradually, not abruptly
  • Natural appearance: Creates organic, flowing patterns
  • Predictable yet varied: Same input gives same output, but looks random
  • Multi-dimensional: Works in 1D, 2D, 3D for complex patterns

How Noise Works:

  • noise() takes one or more coordinates as input
  • Returns a value between 0 and 1
  • Similar input coordinates produce similar output values
  • This creates smooth, continuous curves

1D Noise (One Dimension):

Code example 9 written in javascript. The code follows.
let xoff = 0;  // Offset in noise space

function draw() {
  background(220);
  // Get noise value (0 to 1), scale to canvas height
  let y = noise(xoff) * height;
  ellipse(200, y, 50, 50);
  
  // Increment offset (small increments = smooth movement)
  xoff += 0.01;  // Try different values: 0.001 (very smooth) to 0.1 (more variation)
}

2D Noise (Two Dimensions):
Creates smooth patterns across a 2D surface:

Code example 10 written in javascript. The code follows.
for (let x = 0; x < width; x += 10) {
  for (let y = 0; y < height; y += 10) {
    // Get noise value for this position
    // Scale coordinates down (0.01) for smoother variation
    let bright = noise(x * 0.01, y * 0.01) * 255;
    fill(bright);
    rect(x, y, 10, 10);
  }
}

3D Noise (Three Dimensions):
Adds time dimension for animated patterns:

Code example 11 written in javascript. The code follows.
function draw() {
  for (let x = 0; x < width; x += 10) {
    for (let y = 0; y < height; y += 10) {
      // Third parameter (frameCount) adds time dimension
      let bright = noise(x * 0.01, y * 0.01, frameCount * 0.01) * 255;
      fill(bright);
      rect(x, y, 10, 10);
    }
  }
}

Noise Scale:
The multiplier (like 0.01) controls how quickly noise values change:

  • Small values (0.001): Very smooth, slow variation
  • Medium values (0.01): Balanced smoothness
  • Large values (0.1): More variation, less smooth

Common Patterns:

Code example 12 written in javascript. The code follows.
// Smooth movement
let x = noise(frameCount * 0.01) * width;

// Organic size variation
let circleSize = noise(frameCount * 0.02) * 100 + 50;

// Flowing color
let r = noise(frameCount * 0.01) * 255;
let g = noise(frameCount * 0.01 + 100) * 255;  // Offset for different pattern
let b = noise(frameCount * 0.01 + 200) * 255;

Probability

Probability lets you control how often random events occur. You can create weighted randomness where some outcomes are more likely than others.

How Probability Works:

  • random(100) returns 0 to 99.999...
  • Compare to a threshold to create probability
  • random(100) < 10 is true 10% of the time (when result is 0-9.999)

Basic Probability:

Code example 13 written in javascript. The code follows.
if (random(100) < 10) {
  // 10% chance: happens when random(100) is 0-9.999
  fill(255, 0, 0);  // Red
} else {
  // 90% chance: happens when random(100) is 10-99.999
  fill(0, 0, 255);  // Blue
}

Probability Examples:

Code example 14 written in javascript. The code follows.
// 50% chance
if (random(100) < 50) {
  // Do something
}

// 25% chance
if (random(100) < 25) {
  // Do something
}

// 1% chance (rare event)
if (random(100) < 1) {
  // Do something special
}

Multiple Probability Levels:

Code example 15 written in javascript. The code follows.
let chance = random(100);

if (chance < 10) {
  // 10% chance: rare
  fill(255, 0, 0);
} else if (chance < 40) {
  // 30% chance: uncommon (10-40)
  fill(0, 255, 0);
} else {
  // 60% chance: common (40-100)
  fill(0, 0, 255);
}

Using Probability in Loops:

Code example 16 written in javascript. The code follows.
for (let i = 0; i < 100; i++) {
  if (random(100) < 20) {
    // 20% of items will be red
    fill(255, 0, 0);
  } else {
    fill(0);
  }
  ellipse(i * 10, 200, 20, 20);
}

Probability for Behavior:

Code example 17 written in javascript. The code follows.
function draw() {
  if (random(100) < 5) {
    // 5% chance per frame: add new particle
    particles.push(new Particle(mouseX, mouseY));
  }
}

Important Functions

Function Purpose Example
random(max) Random 0 to max random(100)
random(min, max) Random min to max random(10, 20)
noise(x) Smooth random noise(frameCount * 0.01)
noise(x, y) 2D smooth random noise(x * 0.01, y * 0.01)

Common Patterns

Random Distribution

Code example 18 written in javascript. The code follows.
for (let i = 0; i < 100; i++) {
  let x = random(width);
  let y = random(height);
  let circleSize = random(10, 30);
  ellipse(x, y, circleSize, circleSize);
}

Organic Movement

Code example 19 written in javascript. The code follows.
let xoff = 0;
let yoff = 10000;

function draw() {
  let x = noise(xoff) * width;
  let y = noise(yoff) * height;
  ellipse(x, y, 30, 30);
  xoff += 0.01;
  yoff += 0.01;
}

Probability-Based Behavior

Code example 20 written in javascript. The code follows.
function draw() {
  if (random(100) < 5) {
    // 5% chance: draw red circle
    fill(255, 0, 0);
    ellipse(random(width), random(height), 50, 50);
  } else {
    // 95% chance: draw blue circle
    fill(0, 0, 255);
    ellipse(random(width), random(height), 30, 30);
  }
}
