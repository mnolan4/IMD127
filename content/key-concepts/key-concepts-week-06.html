Week 6: Arrays and Collections - Key Concepts

Learning Outcomes

By the end of this week, you should be able to create and use arrays to store multiple values. You should use array methods to manipulate data. You should create particle systems. You should manage multiple objects efficiently.

Key Concepts

Arrays

Arrays are collections that store multiple values in a single variable. They're essential for managing groups of related data, like positions of multiple objects, colors for a palette, or any list of values.

Why Use Arrays:

  • Store collections: Keep multiple related values together
  • Manage many objects: Track positions, sizes, colors for many shapes
  • Process data: Work with lists of information
  • Dynamic collections: Add and remove items as needed

Creating arrays:
Arrays use square brackets [] and separate values with commas:

Code example 1 written in javascript. The code follows.
let numbers = [10, 20, 30, 40, 50];
let colors = ['red', 'blue', 'green'];
let positions = [100, 200, 300];
let empty = [];  // Empty array

Array Indexing:
Arrays use zero-based indexing, meaning the first element is at index 0, not 1.

Code example 2 written in javascript. The code follows.
let arr = [10, 20, 30, 40, 50];
// Index:   0   1   2   3   4

arr[0]  // = 10 (first element)
arr[1]  // = 20 (second element)
arr[4]  // = 50 (fifth element)

Accessing Elements:

Code example 3 written in javascript. The code follows.
let numbers = [10, 20, 30];
console.log(numbers[0]);  // Prints 10
console.log(numbers[1]);  // Prints 20

// Use in code
ellipse(numbers[0], 200, 50, 50);  // Circle at x=10

Array Length:
The length property tells you how many elements are in the array:

Code example 4 written in javascript. The code follows.
let arr = [10, 20, 30];
console.log(arr.length);  // Prints 3

// Last element is at index (length - 1)
let last = arr[arr.length - 1];  // = 30

Modifying Elements:
You can change array elements by assigning new values:

Code example 5 written in javascript. The code follows.
let arr = [10, 20, 30];
arr[1] = 99;  // Change second element
// arr is now [10, 99, 30]

Array Methods

Array methods are built-in functions that help you work with arrays. They make it easy to add, remove, and manipulate array elements.

Adding Elements:

push() adds an element to the end of the array:

Code example 6 written in javascript. The code follows.
let arr = [];
arr.push(10);  // Add 10 to end
arr.push(20);  // Add 20 to end
// arr is now [10, 20]

Removing Elements:

pop() removes and returns the last element:

Code example 7 written in javascript. The code follows.
let arr = [10, 20, 30];
let last = arr.pop();  // Removes 30, arr is now [10, 20]

Array Length:
length property (not a method, but important):

Code example 8 written in javascript. The code follows.
let arr = [10, 20, 30];
console.log(arr.length);  // 3

Looping Through Arrays:

Method 1: Traditional for loop (most control):

Code example 9 written in javascript. The code follows.
let numbers = [10, 20, 30, 40];

for (let i = 0; i < numbers.length; i++) {
  // i is the index (0, 1, 2, 3)
  // numbers[i] is the value
  ellipse(numbers[i] * 10, 200, 30, 30);
}

Method 2: for...of loop (simpler, when you don't need index):

Code example 10 written in javascript. The code follows.
for (let num of numbers) {
  // num is the value directly (10, 20, 30, 40)
  ellipse(num * 10, 200, 30, 30);
}

When to use which:

  • Use traditional for loop when you need the index
  • Use for...of when you only need the values

Finding Elements:

Code example 11 written in javascript. The code follows.
let arr = [10, 20, 30, 40];
let index = arr.indexOf(30);  // Returns 2 (index of value 30)

Arrays of Colors and random()

Arrays can hold any type of value—not just numbers. Strings work too: color names (e.g. 'red', 'DodgerBlue') or hex strings (e.g. '#ff6b6b'). A common pattern is to define a palette (array of colors) and use p5's random(array) to pick one random element. p5's random() accepts an array and returns one random element—so random(palette) gives you a random color from the palette.

Generative artists often use this instead of fully random RGB: you curate a small set of colors and let the program choose from them, so results stay coherent while still varied.

Code example 11a written in javascript. The code follows.
let palette = ['red', 'blue', 'green', '#ffaa00', 'DodgerBlue'];

function draw() {
  fill(random(palette));  // Pick random color from palette
  ellipse(random(width), random(height), 40, 40);
}

Particle Systems

Particle systems use arrays to manage many objects (particles) at once. Each particle has properties like position, velocity, color, or size. Arrays let you store and update all particles efficiently.

Why Arrays for Particles:

  • Store positions for many particles: let x = [100, 200, 300, ...]
  • Update all particles in a loop
  • Add or remove particles dynamically
  • Create complex visual effects with many elements

Basic Particle System (Parallel Arrays):
Store related data in separate arrays with matching indices:

Code example 12 written in javascript. The code follows.
let x = [];      // x positions
let y = [];      // y positions
let vx = [];     // x velocities
let vy = [];     // y velocities

function setup() {
  createCanvas(400, 400);
  // Create 50 particles
  for (let i = 0; i < 50; i++) {
    x.push(random(width));
    y.push(random(height));
    vx.push(random(-2, 2));
    vy.push(random(-2, 2));
  }
}

function draw() {
  background(220);
  // Update and draw all particles
  for (let i = 0; i < x.length; i++) {
    // Update position
    x[i] += vx[i];
    y[i] += vy[i];
    
    // Wrap around edges
    if (x[i] > width) x[i] = 0;
    if (x[i] < 0) x[i] = width;
    if (y[i] > height) y[i] = 0;
    if (y[i] < 0) y[i] = height;
    
    // Draw particle
    ellipse(x[i], y[i], 10, 10);
  }
}

Particle System with Objects (Better Organization):
Using objects is cleaner than parallel arrays:

Code example 13 written in javascript. The code follows.
let particles = [];

function setup() {
  createCanvas(400, 400);
  for (let i = 0; i < 50; i++) {
    particles.push({
      x: random(width),
      y: random(height),
      vx: random(-2, 2),
      vy: random(-2, 2)
    });
  }
}

function draw() {
  background(220);
  for (let p of particles) {
    p.x += p.vx;
    p.y += p.vy;
    ellipse(p.x, p.y, 10, 10);
  }
}

Common Patterns

Multiple Objects

Code example 14 written in javascript. The code follows.
let x = [100, 200, 300];
let y = [100, 200, 300];
let diameter = [50, 75, 100];

function draw() {
  background(220);
  for (let i = 0; i < x.length; i++) {
    ellipse(x[i], y[i], diameter[i], diameter[i]);
  }
}

Adding/Removing Particles

Code example 15 written in javascript. The code follows.
let particles = [];

function mousePressed() {
  particles.push({x: mouseX, y: mouseY});
}

function draw() {
  background(220);
  for (let i = particles.length - 1; i >= 0; i--) {
    ellipse(particles[i].x, particles[i].y, 10, 10);
    particles[i].y += 1;  // Fall down
    if (particles[i].y > height) {
      particles.splice(i, 1);  // Remove if off screen
    }
  }
}
